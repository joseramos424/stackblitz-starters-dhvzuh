# Gestor de Tareas

Un sistema simple de gestión de tareas desarrollado en Python con SQLite.

## Características

- Agregar tareas
- Ver lista de tareas
- Actualizar tareas
- Eliminar tareas
- Almacenamiento persistente con SQLite

## Requisitos
#
- Python 3.x
- SQLite3

## Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/TU_USUARIO/gestor-tareas.git
```
